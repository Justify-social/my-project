UI Component Autoloading Test

This page provides a quick way to verify that UI component autoloading is working correctly. Click the button below to navigate to the UI Component Debug Tools page.

Verification Steps

Click the button below to open the UI Component Debug Tools
Components should load automatically (no need to click "Start Watching")
Verify that the components appear in the Components tab
Check browser console for logs confirming autoload completion
Open UI Component Debug Tools
